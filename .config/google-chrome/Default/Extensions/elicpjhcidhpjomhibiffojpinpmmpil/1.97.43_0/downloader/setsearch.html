How to set WebWebWeb incl. video search as your browser search

Please read all the instructions and then start with point 1.

1 Click button Click here onto
set search
to add WebWebWeb search to your browser.


2 Click OK Chrome displays a dialog box in which you need to confirm that WebWebWeb is added.



Just click OK. If webwebweb already has been added to list of your searches the OK button is disabled. Click Cancel in this case.

3 Make WebWebWeb default An overview of all search providers is shown. If there's no (Default) next to WebWebWeb move the mouse cursor over WebWebWeb and then click Make Default
4 Ready To search via WebWebWeb just enter your search terms into the URL box of Chrome.
